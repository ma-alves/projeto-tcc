CASOS DE USO ESPECÍFICOS
Criar os seguintes casos para Administrador:
    Administrador - Adicionar consulta -> acessar página de adicionar consulta 
                                    -> selecionar data, hora e valor
                                    -> sistema confirma criação da consulta

                - Apagar consulta -> acessar perfil
                                    -> selecionar horário vago a ser apagado
                                    -> sistema confirma que o horário vago foi apagada

                - Cadastrar psicólogo -> acessar página de cadastro de psicólogo
                                        -> sistema valida permissão para cadastrar
                                        -> registrar nome, crp, e-mail e senha
                                        -> sistema valida e confirma novo cadastro

                - Editar perfil -> acessar página de alteração de perfil
                                -> alterar informações
                                -> sistema valida e salva as novas informações
