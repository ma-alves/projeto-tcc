CASOS DE USO ESPECÍFICOS
    Administrador - Editar consulta -> acessar página de consultas marcadas
                                    -> selecionar consulta
                                    -> editar dados
                                    -> confirmar alterações

    Administrador - Adicionar pacote -> acessar página de adicionar consulta ou pacote
                                    -> selecionar datas, hora e valor
                                    -> sistema confirma criação do pacote
    Usuário - Contratar pacote -> acessar página de agendar consultas
                               -> selecionar pacote
                               -> sistema confirma contratação do pacote
